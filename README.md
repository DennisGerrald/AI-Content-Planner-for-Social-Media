# AI Content Planner for Social Media

An AI-powered content planner to generate social media captions, reel ideas, and hashtags using Google Gemini generative AI.  
This project combines a Node.js + Express backend, MongoDB for scheduling data, and a React frontend for user interaction.

---

## Features

- Generate social media *captions, **reel ideas, and **hashtags* using Gemini AI.
- Store scheduled posts with content and posting dates in MongoDB.
- Simple REST API to manage schedules.
- React UI to enter prompts and view AI-generated results.
- Easily extensible for analytics tracking and multi-platform posting.

---

## Tech Stack

| Layer        | Technology          |
|--------------|---------------------|
| AI Model     | Google Gemini API   |
| Backend      | Node.js, Express    |
| Database     | MongoDB             |
| Frontend     | React (Create React App) |
| Deployment   | GitHub (CI/CD workflow ready) |

---

## Prerequisites

- Node.js v18+
- MongoDB instance (local or cloud)
- Google Gemini API key  
  (Sign up & create credentials at [Google AI Platform](https://ai.google.com/))
